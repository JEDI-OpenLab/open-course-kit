Comprendre les licences

Choisir une licence ouverte pour une ressource pédagogique

Publier une ressource pédagogique en ligne ne suffit pas à la rendre réutilisable. Pour qu’elle puisse être copiée, adaptée, traduite, remixée ou intégrée dans un autre contexte, il faut indiquer clairement ce que les autres ont le droit de faire avec elle.

Cette page propose un repère pratique pour comprendre les principales licences ouvertes et choisir une option adaptée à une ressource pédagogique.

Point de vigilance

Ce guide propose des repères généraux. Il ne constitue pas un conseil juridique professionnel. En cas de doute sur des droits d’auteur, des images, des données personnelles, des marques, des contrats ou des règles institutionnelles, il convient de consulter les services compétents.

Trois distinctions essentielles

Gratuit

Une ressource gratuite peut être consultée sans paiement, mais cela ne signifie pas automatiquement qu’elle peut être copiée, modifiée ou redistribuée.

Libre d’accès

Une ressource librement accessible peut être lue ou téléchargée, mais ses conditions de réutilisation peuvent rester limitées ou non précisées.

Libre de réutilisation

Une ressource sous licence ouverte indique explicitement les droits accordés : copier, adapter, redistribuer, citer, transformer ou intégrer ailleurs, selon les conditions de la licence.

Les licences Creative Commons les plus utiles

Les licences Creative Commons sont souvent utilisées pour publier des contenus pédagogiques : textes, supports de cours, images, fiches, guides, scénarios, exemples, ressources web ou documents d’accompagnement.

Comparatif simplifié des licences Creative Commons
Licence Réutilisation Modification Usage commercial Attribution Cas d’usage recommandé
CC0 Oui Oui Oui Non juridiquement requise Ressources originales que l’on souhaite placer aussi largement que possible dans le domaine public volontaire.
CC BY Oui Oui Oui Oui Ressources ouvertes avec attribution obligatoire de l’auteur ou du projet.
CC BY-SA Oui Oui Oui Oui Ressources ouvertes avec obligation de partager les adaptations sous la même licence.
CC BY-NC Oui Oui Non Oui Ressources dont on souhaite interdire les usages commerciaux, avec prudence car la notion peut être ambiguë.
CC BY-ND Oui Non Oui Oui Ressources que l’on souhaite diffuser sans modification. Peu adapté aux ressources éducatives libres.

Focus : CC0 1.0 Universal

La licence CC0 1.0 Universal permet à un auteur de renoncer, dans toute la mesure permise par la loi, à ses droits patrimoniaux sur une ressource. C’est l’option la plus permissive : elle autorise la copie, la modification, la republication, l’adaptation et la transformation sans demande d’autorisation préalable.

Quand choisir CC0 ?

CC0 est particulièrement adaptée lorsque l’objectif principal est de maximiser la circulation et la réutilisation d’une ressource originale : gabarits, exemples, checklists, fichiers web, modèles de documents, jeux de données pédagogiques ou supports produits spécifiquement pour être repris.

Modèle de mention CC0

## Licence

Sauf mention contraire, les contenus, documents, exemples, fichiers web et ressources originales de cette ressource sont placés sous [CC0 1.0 Universal](https://creativecommons.org/publicdomain/zero/1.0/deed.fr).

Cela signifie que l’auteur renonce, dans toute la mesure permise par la loi, à ses droits patrimoniaux sur ces contenus afin qu’ils puissent être copiés, adaptés, réutilisés, republiés et transformés sans demander d’autorisation.

Les marques, logiciels, services tiers, documents sources importés dans le corpus et ressources citées restent soumis à leurs licences et droits respectifs.

Comment choisir ?

1

Identifier ce que vous avez réellement créé

Une licence ne peut s’appliquer qu’aux contenus pour lesquels vous détenez les droits nécessaires. Commencez par séparer les contenus originaux des images, textes, données, logos, logiciels ou documents tiers.

2

Définir le niveau d’ouverture souhaité

Voulez-vous autoriser les modifications ? Les usages commerciaux ? L’intégration dans d’autres ressources ? L’obligation d’attribution ? Le partage dans les mêmes conditions ?

3

Indiquer clairement la licence

Ajoutez un fichier LICENSE, une section licence dans le README.md et, si possible, une mention de licence sur les pages principales du site ou du document.

Recommandations rapides

Pour une ressource pédagogique très ouverte

Choisir CC0 si vous voulez autoriser la réutilisation la plus large possible sans obligation juridique d’attribution.

Pour une ressource ouverte avec citation obligatoire

Choisir CC BY si vous voulez permettre la réutilisation et la modification, tout en imposant l’attribution.

Pour une logique de commun partagé

Choisir CC BY-SA si vous voulez que les adaptations soient repartagées sous la même licence.

Pour une ressource pédagogique vraiment adaptable

Éviter autant que possible les clauses ND, qui interdisent les modifications, car elles limitent fortement l’adaptation pédagogique.

Droits tiers : ce qui reste à vérifier

Une ressource peut être publiée sous licence ouverte tout en contenant certains éléments qui ne le sont pas. Dans ce cas, il faut le signaler clairement et éviter de laisser croire que tout le contenu est réutilisable de la même manière.

  • Images, icônes, photographies et captures d’écran
  • Extraits de livres, articles, sites web ou documents institutionnels
  • Logos, marques, noms de produits et identités visuelles
  • Logiciels, bibliothèques, plugins, polices et composants tiers
  • Données personnelles, travaux étudiants, témoignages ou informations sensibles
  • Documents importés dans un corpus documentaire ou une base de connaissances

Et si le projet contient du code ?

Les licences Creative Commons sont adaptées aux contenus pédagogiques, mais elles ne sont généralement pas les plus adaptées au code informatique. Si une ressource contient une webapp, des scripts, des composants JavaScript ou du code réutilisable, il est préférable d’ajouter une licence logicielle dédiée, par exemple MIT, Apache 2.0 ou GPL, selon le niveau d’ouverture et les obligations souhaitées.

Une solution simple consiste à distinguer clairement les deux niveaux : une licence pour les contenus pédagogiques et une licence pour le code.

À retenir

La bonne licence dépend de l’objectif de réutilisation. Pour favoriser au maximum la circulation d’une ressource originale, CC0 est l’option la plus ouverte. Pour garantir l’attribution, CC BY est souvent un bon équilibre. Pour imposer le partage dans les mêmes conditions, CC BY-SA peut être pertinente. Dans tous les cas, il faut vérifier les droits des contenus tiers avant de publier.
